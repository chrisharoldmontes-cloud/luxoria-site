# Luxoria - Événement Prestigieux

Site web officiel de l'événement Luxoria, un rassemblement d'exception pour professionnels, entrepreneurs et innovateurs.

## 📅 Informations
- **Date** : 15-17 Juin 2026
- **Lieu** : Brazzaville - ESGAE
- **Participants** : 500+

## 🎯 Fonctionnalités
- Présentation de l'événement
- Programme détaillé sur 3 jours
- Galerie photo interactive
- Formulaire de réservation
- Informations sur les intervenants

## 🛠 Technologies utilisées
- HTML5
- CSS3 (avec animations)
- Bootstrap 5
- JavaScript (ES6)
- Font Awesome (icônes)

## 📱 Responsive Design
Le site est entièrement responsive et s'adapte à tous les appareils (mobile, tablette, desktop).

## 🚀 Déploiement
Ce site est déployé sur GitHub Pages pour un hébergement gratuit et fiable.
